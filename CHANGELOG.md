# Changelog

All notable changes to Midnight Fetcher Bot will be documented in this file.

## [1.0.0] - 2025-11-10

### Initial Release

First production release of Midnight Fetcher Bot.

#### Core Features
- Multi-address mining with up to 200 addresses
- High-performance Rust hash server with parallel processing
- Real-time web dashboard for monitoring
- Secure wallet creation and management
- Automatic address registration
- Solution receipt logging and history
- Windows setup automation

#### System Requirements
- Windows 10/11 (64-bit)
- Node.js v20.11.1 or higher
- 4GB RAM minimum
- 500MB free disk space

#### Known Issues
None at release time.
